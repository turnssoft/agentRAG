New Employee IT Onboarding Checklist

GI
Created by Gary IT, Last modified on July 28, 2025

Onboarding Target: New Store Manager | Employee Name: Jane Smith | Start Date: August 5, 2025

Phase 1: Pre-Arrival (Due by EOD August 4)

Tasks to be completed before the employee's first day.

Account Provisioning

  • Create Active Directory account (j.smith).
  • Create email account (jane.smith@pizzaboys.com).
  • Add user to 'StoreManagers' security group.
  • Provision access to 'DoughMaster POS' backend portal with Manager-level permissions.

Hardware Setup

  • Image new Dell laptop with standard corporate image v3.2.
  • Install 'DoughMaster POS' client and 'InventoryManager' client.
  • Provision new company smartphone (iPhone 15).
  • Assign hardware assets in inventory system.

Phase 2: Day One (To be completed on August 5)

Tasks for the employee's first day in the office.

Security First! Do not provide passwords directly. Use the secure temporary password system and force a password change on first login.

Welcome & Setup

  • Deliver laptop and smartphone to the new employee.
  • Assist with first-time login and mandatory password reset.
  • Guide user through setting up Multi-Factor Authentication (MFA) on their smartphone.
  • Provide a brief tour of the PizzaNet intranet and helpdesk portal.

Phase 3: First Week (Due by EOD August 12)

Follow-up tasks to ensure the employee is fully integrated.

Training & Confirmation

  • Confirm employee has completed the mandatory 'Cybersecurity Awareness' training module.
  • Schedule a 30-minute follow-up meeting to answer any IT-related questions.
  • Verify that access to all required systems is working as expected.
